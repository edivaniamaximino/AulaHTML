<h1>LogFull - Website </h1>

Este projeto apresenta um site institucional fictício da LogFull, destacando serviços, informações da empresa e uma navegação completa entre páginas. Desenvolvido em HTML e CSS, ele serve como exercício prático para aplicar conceitos essenciais de desenvolvimento web, com foco em design limpo, usabilidade e boa organização de código, representando uma base sólida para sites estáticos.

<h2>Objetivo do Projeto</h2>

Criar um site institucional simples, elegante e funcional que possa ser utilizado como base para projetos de logística, transporte, supply chain ou empresas que desejam apresentar seus serviços de forma clara e profissional.

<h2>Tecnologias Utilizadas</h2>

HTML5 para estruturação das páginas

CSS3 para estilização, responsividade, grids e componentes visuais

Layout responsivo para mobile e desktop

Navegação clara e organizada com menu fixo

<h2>Funcionalidades do Projeto</h2>

Página inicial com hero section e apresentação da marca

Sessão de serviços com cards dinâmicos e imagens

Página “Sobre” com foco na proposta de valor da LogFull

Formulário de contato funcional (HTML)

Design moderno com sombras, animações suaves e hierarquia visual

Cabeçalho fixo, rodapé institucional e imagens otimizadas

Estrutura pronta para evoluir para backend ou integração com APIs futuramente


<h2>Finalidade</h2>

Estudos de HTML e CSS

Protótipos de sites institucionais

Base para futuros projetos web

Portfólio profissional
